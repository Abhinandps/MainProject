Before Launching WebApp You must install in your system,
	
	* WampServer
	* python 


Then Open the Code Editor VScode -> choose Terminal then,
1) clone the project from github 
   
   git clone https//github...copypath......

2) check the terminal in current directory

3) Choose Terminal do the below steps

STEP 1
******
Windows
-------
py -3 -m venv venv

------
python3 -m venv venv



STEP 2
******
Activate the environment
---------------------

-------
venv\Scripts\activate


Linux
-----
. venv/bin/activate



STEP 3
******
pip install Flask

pip install Flask-MySQLdb

pip install xlwt

pip install OpenCV-contrib-python
